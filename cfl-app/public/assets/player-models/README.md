# Player Model Assets

Drop temporary player model renders here while the game is using user-generated assets.

Required source format:

- transparent PNG
- `1024x1024`
- filename: `<personId>.png`
- example: `person_007869.png`

Recommended prompt shape:

```text
Transparent-background full-body or thighs-up college football player render, front three-quarter pose, dramatic stadium key light, premium sports video game menu asset, accurate team uniform colors, sharp fabric detail, no text, no logo hallucinations, no background, no border, centered subject, 1024x1024.
```

Use the `personId` from the roster data so code can map assets deterministically.
