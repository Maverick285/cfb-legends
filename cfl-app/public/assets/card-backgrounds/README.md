# Card Background Assets

Drop optional team/player feature-card backing plates here.

Recommended source format:

- PNG or JPG
- `1536x864`
- filename: `<programId>-player-card-bg.png`
- example: `program_vanderbilt-player-card-bg.png`

These should be environmental plates, not full UI mockups: stadium light, smoke, team-color texture, field/tunnel atmosphere, or locker-room treatment. Do not bake player names, ratings, buttons, or table UI into the image.
