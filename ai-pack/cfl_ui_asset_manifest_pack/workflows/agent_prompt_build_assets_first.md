# Agent Prompt — Build Assets First

You are not building full feature screens yet.

Your job is to implement the UI asset/component foundation for College Football Legends.

## Required order

1. Implement design tokens.
2. Implement team brand token schema.
3. Implement global shell components.
4. Implement Panel, Button, Sidebar, Header, Footer/Ticker.
5. Implement rating/star/grade/progress widgets.
6. Implement asset loading conventions.
7. Add placeholder assets where final art does not exist.
8. Add visual tests for the components.

## Rules

- Do not hardcode team-specific colors in components.
- Do not bake dynamic text into image assets.
- Do not build one-off screen layouts.
- If an asset is missing, create a clearly named placeholder and record it in the manifest.
- Every component must support theme/team tokens.

## Final report

List:

- files changed
- assets added
- components added
- manifest updates
- commands run
- visual test artifacts
- known missing final art
