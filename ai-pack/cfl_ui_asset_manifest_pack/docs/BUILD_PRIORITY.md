# Build Priority

## Phase 0 — Inventory and baselines

1. Put all reference images in `specs/reference-images/`.
2. Annotate each image using `docs/ANNOTATION_GUIDE.md`.
3. Fill `manifests/screen_asset_map.csv` for each screen.
4. Fill `manifests/ui_asset_manifest.csv` with reusable assets.

## Phase 1 — Global graphics

Build or implement:

1. game logo lockups
2. app background texture
3. header background
4. sidebar background
5. footer/ticker background
6. standard panel skin
7. raised panel skin
8. table container skin
9. right detail panel skin
10. divider system
11. button skins or CSS button variants
12. icon family

Do not build screen-specific art until the global component and panel system exists.
