# Static vs Dynamic Rules

## Never bake these into images

- school names
- player names
- coach names
- records
- rankings
- scores
- grades
- percentages
- budget numbers
- NIL values
- schedule rows
- table text
- button labels
- chart values
- star counts
- active states
- selected row states

## Usually static

- logo art
- icon art
- decorative corners
- subtle texture overlays
- static background texture
- general frame art
- signed stamp
- base stadium diagram
- base uniform mannequin
- helmet base artwork

## Usually hybrid

- score banner
- matchup card
- recruiting card
- NIL budget card
- postgame grade card
- stadium expansion card
- school identity preview card

## If uncertain

Default to dynamic/programmatic.

A static image is acceptable only when it adds real visual quality and does not trap future dynamic data inside a flattened graphic.
