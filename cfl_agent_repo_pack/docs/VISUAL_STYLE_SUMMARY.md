# Visual Style Summary — College Football Legends

## North Star
A premium, dark, dense, data-heavy college football dynasty management sim. It should feel like Football Manager crossed with a modern American college football broadcast package and an athletic department command center.

## Mood
- dark metallic UI
- serious sports-management tone
- premium broadcast graphics
- dense cards and tables
- school identity everywhere
- prestige, rankings, recruiting, NIL, staff, facilities, uniforms, and stadium as core systems

## Global Visual Rules
- Use dark charcoal/navy/black surfaces.
- Use team colors for active states, headers, buttons, highlights, and selected rows.
- Use condensed uppercase athletic typography for headings/buttons.
- Use compact sans-serif for metadata and tables.
- Use subtle borders, gradients, inner highlights, and panel shadows.
- Avoid generic web-dashboard minimalism.
- Avoid bright white backgrounds.
- Avoid overly rounded cards.
- Avoid hardcoded one-off styling.

## Core Page Archetypes
1. Dashboard / Team Hub
2. Sidebar Management Layout
3. Full Table Layout
4. Cinematic Event Layout
5. Campus Map / Facilities Layout
6. Builder Workbench Layout
7. Wizard Creator Layout
8. Skill Tree / Progression Layout
9. Postgame Report Card Layout
10. Macro Hub Tile Layout

## Important Screens
- Dashboard
- Recruiting Board
- Signing Day
- NIL Management
- Stadium Builder
- Uniform Creator
- School Creator
- Coaching Upgrades
- Game Report Card
- Campus Facilities Map
- Team Management Hub

## Design System Requirement
Every screen must be assembled from shared shells, templates, components, design tokens, team brand tokens, and typed data.

Do not create one-off mockups.
