# Dashboard Visual Spec

## Page Archetype
Dashboard / Team Hub

## Purpose
Weekly command center showing current team status, next game, rankings, recruiting, staff, schedule, tasks, and Continue action.

## Required Regions
- App header with team identity
- Left sidebar or top nav
- Main dashboard grid
- Current matchup card
- Team overview/rating card
- Schedule/upcoming games card
- Recruiting overview card
- Staff overview card
- Rankings or news card
- Bottom ticker/footer
- Continue or Play Week CTA

## Visual Hierarchy
1. Team identity and current week at top.
2. Current matchup and team overview as primary cards.
3. Supporting cards in dense grid.
4. Bottom ticker creates living world feel.

## Required Components
- AppHeader
- SidebarNav or TopTabNav
- Panel
- StatTile
- RatingRing
- ScheduleRow
- RankingRow
- BottomTicker
- ActionButton

## Fail Conditions
- Missing current week/season.
- Missing next opponent.
- Missing team logo/name/record/rank.
- Sparse layout.
- Continue CTA not obvious.
- Cards not aligned to grid.
