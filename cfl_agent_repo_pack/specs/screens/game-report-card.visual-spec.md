# Game Report Card Visual Spec

## Page Archetype
Postgame Report / Evaluation

## Purpose
Shows final score, game grades, key stats, top performers, game impact, development changes, and next opponent.

## Required Regions
- App header or postgame header
- Score banner
- Quarter line score
- Overall game grade
- Unit grade cards
- Key stats comparison
- Top performers
- Key moments
- Game impact
- Player development
- What's next CTA

## Required Components
- PostgameShell
- ScoreBanner
- QuarterLineScore
- ReportGradeCard
- ComparisonTable
- TopPerformerCard
- ImpactList
- KeyMomentsPanel
- PlayerDevelopmentRow
- NextOpponentCard

## Fail Conditions
- Score banner missing or weak.
- Grades do not dominate visually.
- No consequences/impact.
- Stats hardcoded in component.
- Next opponent CTA missing.
